# `Control` Component
The `Control` component is a customizable form control that renders a field based on its type and checks its conditions. This component is used to manage form input data.

### Storybook
You can view the `Control` component in the Storybook.

### Supported Types
- `CheckboxMultiple`
- `CheckboxSingle`
- `DatePickerRange`
- `DatePickerSingle`
- `Input`
- `Mentions`
- `Number`
- `Radio`
- `Rate`
- `SelectMultiple`
- `SelectSingle`
- `SelectTag`
- `SliderSingle`
- `SliderRange`
- `Switch`
- `Textarea`
- `TimePickerRange`
- `TimePickerSingle`
